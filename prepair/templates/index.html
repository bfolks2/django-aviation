PrepAir.com

Or at least that could be the name of this site if I had purchased that domain...

This site is built to allow users to get up-to-date flight simulating information. Anytime a user searches for an airport, real-time weather information is displayed, by use of an external API. Additionally, if the airport searched by a user does not yet exist in this site's internal database, all additional static data (i.e. name, runway info, communication/frequencies, etc) is then permanently saved into the internal database using this site's own REST API. The size of the internal airport database grows each time a user searches for a new airport. Users can search across the internal database of airports for all relevant information, as well as post their own specific reports, or even comment on other user's experiences.

Powered by Python/Django and Ember.js
